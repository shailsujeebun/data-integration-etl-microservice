# Screenshot Checklist

Add these screenshots before sharing with recruiters:

1. `dashboard.png` - dashboard overview with recent jobs and status cards.
2. `sources.png` - configured sources list with type/auth indicators.
3. `job-detail.png` - one run detail with error log rows visible.
4. `trigger-flow.png` - before and after manual trigger.

Tips:
- Use 1366x768 or 1440x900 resolution.
- Redact any sensitive URLs or credentials.
- Prefer light compression (PNG or high-quality JPG).
